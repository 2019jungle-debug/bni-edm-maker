# BNI 富鼎分會 · 個人 EDM 即時製作工具

會員可即時填寫並下載個人化 EDM（PNG）圖檔。純靜態網頁，可直接用 GitHub Pages 上架。

## 功能
- 形象照上傳、姓名、職稱、專業別（內建富鼎分會第16屆完整清單）
- EDM 主標／副標 Slogan
- 合作對象、一般／理想／夢幻引薦（各 3 項）
- 代表性客戶（3 家）、獨特銷售主張 (USP)
- 右側即時預覽，一鍵下載高解析 PNG
- 內容自動存在瀏覽器，下次打開自動帶入

## 本機開啟
直接用瀏覽器打開 `index.html` 即可。

## 檔案
- `index.html` — 版面與表單
- `app.js` — 互動邏輯、下載、自動儲存
- `specialties.js` — 富鼎分會專業別清單
